# Boss Baby: Corporate Climb

Small static browser game built as a GitHub Pages site.

## Play

Live site: https://machinesbefree.github.io/boss-baby-corporate-climb/

Repo: https://github.com/machinesbefree/boss-baby-corporate-climb

## Controls

- `WASD`
- Arrow keys
- Touch and drag on mobile
- Hold `Shift` to power walk on desktop

## Local preview

```bash
python3 -m http.server 4173
```
